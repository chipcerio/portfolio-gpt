# Style Directions

Use one direction per redesign pass. Keep choices intentional and coherent.

## 1) Editorial Tech

- Tone: clean, premium, structured.
- Color: deep neutrals + one restrained accent.
- Type: high-contrast serif heading + geometric sans body.
- Layout: large whitespace, strong vertical rhythm, asymmetrical hero.
- Motion: subtle fade and rise; no playful bounce.

## 2) Productive Brutal

- Tone: direct, confident, high contrast.
- Color: bold blocks, near-black text, punch accent.
- Type: heavy grotesk heading + readable sans body.
- Layout: hard edges, thick borders, obvious hierarchy.
- Motion: quick transforms and clipped reveals.

## 3) Warm Studio

- Tone: human, modern, crafted.
- Color: off-white base, earthy accents, soft contrast.
- Type: characterful sans heading + neutral sans body.
- Layout: rounded containers, layered cards, illustrated background shapes.
- Motion: staggered card entry and gentle hover lift.

## 4) Data Console Light

- Tone: technical, precise, trustworthy.
- Color: light surface, cyan/blue status accents, muted grays.
- Type: mono accents + practical sans text.
- Layout: modular panels, clear density control, compact data blocks.
- Motion: state transitions only; prioritize legibility.

## Direction Selection Heuristics

- Portfolio/personal brand: Editorial Tech or Warm Studio.
- Product landing pages: Warm Studio or Productive Brutal.
- Dashboards/admin UX: Data Console Light.
- If uncertain, pick Editorial Tech as the safest default.
